# UI Icon Assets

The PNG files in this directory are the packaged runtime icon set. They are 96 x 96 RGBA exports intended to render at a 20-40 logical pixel size.

- `add`, `remove`, `help`, `close`, `loading`, `arrow-left`, and `arrow-right` use geometry from Lucide 0.468.0. The pinned upstream URLs, ISC license, and SHA-256 hashes are recorded in `manifest.ron`.
- `full-color-badge` and `missing` are deterministic MyBevy project-owned assets generated by `scripts/generate-ui-icons.ps1`.
- White transparent icons are `monochrome_tintable`; runtime state tint is permitted only for that policy.
- `full_color` icons must render with `Color::WHITE`. Applying theme or state tint to them is a framework error and is intentionally ignored.

Regenerate from the repository root:

```powershell
.\scripts\generate-ui-icons.ps1
```

After regeneration, update `manifest.ron` only when every changed image has been reviewed and its new hash is intentional.
