# 方圆灵构技术文档

这个目录记录方圆灵构从世界观概念落到工程实现时需要遵守的基础技术路线。

方圆灵构不是单一渲染功能，也不是单一玩法系统。它会同时影响家园、装备、技能、NPC、天道自演、对象资源构建、蓝图审核、运行时渲染、LOD、Bake 和资源加载。

世界观层面的定义仍放在 [世界观/方圆灵构](../世界观/方圆灵构.md)。本目录只记录“如何把方圆灵构做成可扩展技术底座”。

## 当前文档

- [方圆对象资源构建与渲染技术路线](./方圆对象资源构建与渲染技术路线.md)：记录方圆对象资源、runtime primitive、Prefab、Chunk、Bake、实例化渲染、统一材质、预算、LOD、多人热点和阶段开发路线。
- [方圆程序动画与技能表现技术路线](./方圆程序动画与技能表现技术路线.md)：记录装备和技能表现的程序动画、VFX recipe、技能模板、规则层/个性层、同步、LOD、审核和试炼场验收路线。
