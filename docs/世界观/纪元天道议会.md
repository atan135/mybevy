# 纪元天道议会

## 文档定位

本文记录中后期玩家组织影响世界走向的世界观设计口径。

本文是设计文档，不代表宗门、议会、投票、世界规则变更、剧情分支或合服方向选择已经实现。落地时应拆分为玩法规则、服务端协议、UI 流程、运营配置和安全审计等具体设计。

## 核心概念

当一个世界运营到中后期，玩家会逐渐形成宗门、帮派、联盟、散修盟等组织。强大的组织不应只停留在排行榜和领地占有上，而应获得有限的世界影响权。

这类机制不应表达为玩家直接改游戏设定，而应表达为：

- 诸宗议会。
- 天道议盟。
- 纪元法会。
- 上界仙庭。
- 界主制度。
- 宗门共议。

世界观口径是：高活跃、高贡献的组织获得干涉天道、推动纪元走向的资格。

## 设计原则

玩家组织不直接成为策划，也不能无限制修改底层规则。

更合适的设计是：系统在每个纪元给出有限的天道议题，拥有资格的组织和角色在这些议题中选择立场、投入贡献、推动事件，并最终影响下个纪元的世界规则、剧情走向或合服方向。

基本原则：

- 有边界的世界影响权。
- 强影响，但不毁灭其他玩家体验。
- 领头人的权力来自组织成员和世界贡献。
- 普通成员需要能感知并参与组织决策。
- 散人和小组织不能被完全排除。
- 结果应进入纪元记录，形成可追溯历史。

## 权力来源

议会资格不应只由宗主身份决定。权力应来自组织在当前纪元中的真实表现。

可参考因素：

- 宗门或组织活跃度。
- 组织成员贡献总量。
- 世界事件贡献。
- 领地、据点或资源点占有。
- 职业/流派代表性。
- 称号和纪元荣誉。
- 成员授权和内部投票。
- 跨宗门联盟支持度。

组织领头人可以拥有提案权、代表票或议席，但这些权力需要由组织整体贡献和成员授权支撑。

## 天道议题

每个纪元中后期，系统可以开启一批天道议题。议题不应是自由文本，而应是系统配置好的若干可选世界走向。

议题主要分为三类。

### 世界规则议题

世界规则议题影响下个纪元的资源、玩法压力、区域状态或跨界成本。

示例：

- 火脉复苏：下个纪元火属性资源增加，火系流派成长机会增多。
- 地脉稳固：安全区和据点防御增强，建筑、防御、采集类玩法权重提高。
- 风门大开：探索、远行、跨界传音和跨界支援成本降低。
- 水泽归元：治疗、净化、回流保护和新人恢复机制增强。

### 剧情走向议题

剧情走向议题影响世界事件、NPC 阵营、区域命运和后续剧情分支。

示例：

- 封印魔渊：世界危险度降低，但高阶稀有资源减少。
- 打开魔渊：高阶材料和特殊称号机会增加，但危险事件变多。
- 扶持仙盟：秩序增强，宗门战争和野外冲突受到更多限制。
- 支持散修：个人玩法增强，大型宗门的部分统治权受限。

### 合服方向议题

合服方向议题决定当前世界在下一纪元更倾向进入哪类上界、中界或归并世界。

示例：

- 飞升战争上界：更多 PVP、领地战、宗门冲突和据点争夺。
- 飞升探索上界：更大地图、秘境、远行事件和世界 Boss。
- 飞升资源上界：采集、锻造、交易、建设和资源竞争增强。
- 留守本界：压力较低，发展稳定，但高级玩法开放较慢。

## 推荐流程

一次纪元议会可以按以下流程展开：

1. 纪元中期，系统公布若干天道议题。
2. 符合条件的宗门、联盟或散修组织获得议席、提案权或投票权。
3. 宗主、盟主、长老或界主选择组织立场。
4. 普通成员可以支持、反对、授权、撤销授权或参与组织内部投票。
5. 不同立场触发对应世界事件、任务或阵营活动。
6. 纪元末根据投票、贡献和事件结果结算最终走向。
7. 胜出方向影响下个纪元的世界规则、剧情分支或合服去向。
8. 参与组织和角色获得对应称号、纪元记录和非破坏性奖励。

## 组织领头人的边界

组织领头人可以推动世界方向，但不能直接破坏底层公平性。

不应允许的行为：

- 直接削弱某类普通玩家。
- 关闭基础玩法。
- 取消所有安全区。
- 永久毁掉角色资产。
- 永久封锁低活跃或回流玩家上升路径。
- 让最大组织无限垄断所有议题。
- 让未参与玩家完全承担失败代价。

更适合允许的行为：

- 选择资源倾向。
- 推动剧情阵营。
- 改变世界事件强度。
- 影响下个纪元地图开放方向。
- 影响合服去向。
- 决定部分领地、据点或世界 Boss 的争夺规则。
- 影响跨界交流、支援或交易的成本。

## 防止大组织垄断

该系统最大的风险是少数大宗门长期绑架世界走向。需要从规则层面限制垄断。

可选机制：

- 单组织投票权设置上限。
- 多个小组织可以组成临时联盟。
- 散人阵营拥有独立代表权。
- 成员可以撤销对领头人的授权。
- 组织不能连续主导所有议题。
- 议题结果需要投票和世界事件贡献共同决定。
- 反方阵营也获得补偿任务、纪元称号或回流路径。
- 不同议题按职业、流派、区域或贡献类型拆分代表权。

## 与纪元更替的关系

纪元天道议会应和纪元更替绑定。

议会负责在纪元中后期制造目标、冲突和选择；纪元结算负责把这些选择转化为下个纪元的世界状态。

这可以让玩家感觉：

- 宗门战争不是孤立玩法。
- 世界事件不是一次性活动。
- 合服方向不是运营后台直接决定。
- 排行榜和领地结果能进入世界历史。
- 自己所在组织真的改变了世界。

## 与称号和历史记录的关系

议会结果应进入角色、组织和世界历史。

可以记录：

- 某纪元的主导宗门。
- 某议题的胜出立场。
- 关键世界事件贡献者。
- 反方阵营代表。
- 最终合服或飞升方向。

可以授予：

- 界主类称号。
- 议席类称号。
- 纪元法会参与称号。
- 具体剧情阵营称号。
- 世界事件贡献称号。

称号第一阶段可只作为身份、展示和交互条件，不必直接提供战斗属性。
