# 世界观文档

这个目录记录游戏世界观、世界分层、纪元更替、飞升留界、跨界社交和修行体系相关设计。

## 当前文档

- [世界观总览](./世界观总览.md)：记录当前修行世界、世界层级、纪元更替、飞升留界、跨界社交、安全区和四属性体系的基础口径。
- [纪元天道议会](./纪元天道议会.md)：记录中后期宗门、联盟、散修组织通过诸宗议会、天道议盟、纪元法会等机制影响世界规则、剧情走向和合服方向的设计口径。
- [方圆灵构](./方圆灵构.md)：记录赛博修仙世界中，以球体和正方体为底层法则，通过 AI 生成家园、装备和技能蓝图的设计口径。
- [方圆灵构蓝图规则](./方圆灵构蓝图规则.md)：记录方圆灵构家园原型第一阶段使用的本地 RON 蓝图格式、生成限制和 Codex 生成提示词。
- [天道自演](./天道自演.md)：记录世界/天道主动调用 AI 生成 NPC、场景道具、装饰、异象和弱互动内容的设计口径。
- [灵名溯源](./灵名溯源.md)：记录角色允许重名时，如何通过稳定身份、排行榜同名提醒和真身标记降低冒名误认风险。
- [天道灵压](./天道灵压.md)：记录区域玩家聚集过多时，世界通过灵压、群修天劫和界域震荡进行分流的设计口径。
- [因果留影与风门远行](./因果留影与风门远行.md)：记录玩家分享世界见闻，并通过天道核验和稳定界域锚点进行高频远行的设计口径。
- [背包与须弥空间](./背包与须弥空间.md)：记录随身背包和高阶储物空间的道具存储、远程取物、限制和继承规则。
- [玩法体系设想](./玩法体系设想.md)：记录基于当前世界观拓展出的个人、组织和纪元长期玩法池。
