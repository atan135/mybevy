# 背包与须弥空间

## 文档定位

本文记录道具携带、随身背包和高阶储物空间的世界观设计口径。

本文是设计文档，不代表背包、仓库、远程取物、空间限制、道具分类或储物权限已经实现。落地时应同步拆分为道具系统、背包容量、储物空间权限、服务端校验和 UI 交互规则。

## 核心概念

道具存储分为两类：

- 随身携带：基础背包，随时可用，但容量有限。
- 须弥空间：高阶储物能力，可以无视距离取用，但有获得门槛和存储限制。

背包解决日常携带；须弥空间解决高阶角色的远程储物和长期积累。

## 随身背包

随身背包是角色基础道具空间。

特点：

- 没有获得门槛。
- 没有距离成本。
- 随时可取用。
- 容量有限。
- 适合常用道具。

适合放入：

- 战斗消耗品。
- 常用材料。
- 临时采集物。
- 任务临时物品。
- 常用装备。
- 需要快速使用的道具。

世界观上，随身背包可表现为腰囊、随身法器、基础储物袋或灵机挂件。

## 须弥空间

须弥空间不是传统游戏仓库，而是高阶空间能力。

特点：

- 需要通过修行、职业、称号、任务、道具或世界事件解锁。
- 可以无视距离访问。
- 容量通常高于随身背包。
- 可以随角色跨世界或纪元迁移。
- 不是所有道具都能放入。

须弥空间可理解为角色和天道之间建立的高阶储物通道。角色不需要回到固定仓库，也能从远处调取已存入的物品。

可选命名：

- 须弥空间。
- 洞天藏。
- 灵机库。
- 袖里乾坤。
- 芥子藏界。

本文优先使用“须弥空间”。

## 获得门槛

须弥空间不应默认开放。

可通过这些途径解锁或扩展：

- 职业/流派学习。
- 四属性掌握达标。
- 完成空间相关试炼。
- 获得特定称号。
- 获得可成长储物法器。
- 参与世界事件。
- 纪元奖励。

这样可以让远程储物成为成长目标，而不是普通仓库的替代皮肤。

## 存储限制

须弥空间可以便利，但不能绕过玩法规则。

不适合放入：

- 活体，除非后续设计灵兽袋或洞天生态。
- 不稳定能量体，例如即将爆发的火核、天劫残片。
- 区域绑定资源，例如领地核心、据点旗、世界事件信物。
- 强任务追踪物品。
- 超出空间承载上限的巨大灵构。
- 被天道标记、诅咒、污染或交易锁定的物品。
- 需要保持物理位置的场景道具。

部分高价值资源可允许放入，但需要封印、冷却、权限或额外消耗。

## 与四属性的关系

须弥空间可以和四属性联动：

- 地：提升承载上限，适合矿石、建筑材料、重型道具。
- 火：适合锻造材料、炉心和转化物，但不稳定物品限制更严。
- 水：提升保存能力，适合药材、灵液、丹药和易损物。
- 风：提升远程取用、快速调度和跨区域访问能力。

背包作为基础空间，不必强依赖四属性；须弥空间可以通过四属性形成差异化成长。

## 与职业和流派的关系

职业/流派可影响须弥空间的用途：

- 探索：降低远程取物限制，提升跨区域调度能力。
- 锻造：提升矿石、装备胚子、炉心类物品存储能力。
- 医术：提升药材、丹药、灵液保存能力。
- 建造：提升方圆灵构材料、家园构件存储能力。
- 阵法：增加封印格、隔离格或空间稳定度。

这可以让储物空间不只是容量数值，而是修行路径的一部分。

## 与纪元继承的关系

随身背包和须弥空间都应随角色迁移，但可以有不同校准规则。

- 随身背包随角色直接迁移。
- 须弥空间随角色迁移，但在新世界可能需要重新校准访问权限、空间稳定度和可取用范围。
- 区域绑定、纪元绑定或世界事件绑定物品不一定随角色完整迁移。
- 违规或超限物品应进入封存状态，而不是直接删除。

这样能保护玩家长期投入，同时避免旧世界物品破坏新世界规则。

## 平衡原则

背包和须弥空间需要遵守这些原则：

- 背包免费便利，但容量有限。
- 须弥空间能力更强，但需要成长门槛。
- 远程取物不能绕过战斗、任务、交易和区域规则。
- 某些道具必须保持位置绑定。
- 超限物品优先封存，不直接清空。
- 储物能力应和四属性、职业、称号和纪元规则联动。

## 待补充设计

后续需要继续补充：

- 背包容量规则。
- 须弥空间解锁条件。
- 远程取物限制和冷却。
- 可存储与不可存储道具分类。
- 超限封存规则。
- 与交易、任务、战斗和家园系统的边界。
