# Gameplay 系统文档

这个目录记录 game layer 的玩法系统设计。它不归属于具体场景框架，也不归属于 UI 框架，而是承载角色、机器人、移动、行为、采集、寻路、交互动作和同步规则等具体玩法能力。

## 文档边界

- `docs/scene/` 记录场景生命周期、资源加载、相机、root、spawn、anchor、trigger 和场景切换。
- `docs/ui/` 记录 UI 结构、输入路由、控件、布局和主题。
- `docs/gameplay/` 记录场景进入之后，角色和玩法对象如何移动、执行行为、同步状态、响应输入和参与规则。

## 当前专题
